PowerUp Portal -- Windows install guide
========================================

Everything you need is in this folder. No separate downloads.
Two double-clicks total to set up. Daily use is one double-click.


Step 1.  First-time setup  (one time, ~3 min)
----------------------------------------------
Double-click   install.bat

A black window opens, shows progress, then says "Setup complete!".
Press any key to close.

What it does behind the scenes:
   * If Python isn't installed yet, silently installs Python 3.11
     from the bundled python-3.11.9-amd64.exe (no separate download).
   * Creates a private Python environment in .venv\
   * Downloads required libraries (pandas, customtkinter, etc.)


Step 2.  Run the app  (every time you want to use it)
------------------------------------------------------
Double-click   run-PowerUp-Portal.bat

The PowerUp Portal window opens with 4 tabs (M1 / M2 / M3 / Agreement).


Troubleshooting
---------------
Setup hangs at "Installing Python..."
   --> Wait ~60 seconds. Microsoft's installer is silent and slow.

"Could not install Python automatically"
   --> Double-click python-3.11.9-amd64.exe yourself.
       In the installer, tick the box [x] Add python.exe to PATH.
       Click "Install Now". When it finishes, re-run install.bat.

Windows blocks the .bat with "this app might be harmful"
   --> Right-click the .bat, Properties, tick [x] Unblock, OK.
       Then double-click again.

Nothing happens when I double-click run-PowerUp-Portal.bat
   --> Run install.bat first. It must complete before the launcher works.

App opens then crashes
   --> A clear error dialog appears with the full traceback.
       Screenshot and share with Krutang.

Anything else
   --> Ping Krutang with a screenshot.
