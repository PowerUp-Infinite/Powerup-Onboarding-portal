PowerUp Portal -- macOS install guide
======================================

Everything you need is in this folder. No separate downloads.
Two right-clicks total to set up. Daily use is one double-click.


Step 1.  First-time setup  (one time, ~3 min)
----------------------------------------------
1. ** First time only -- bypass Gatekeeper: **
      Right-click   install.command
      Click   Open
      In the warning dialog, click   Open   again
   macOS remembers this choice.

2. install.command runs in a Terminal window:
      * Looks for Python 3.9+ (most modern Macs already have it)
      * If missing, silently installs Python 3.11 from the bundled
        python-3.11.9-macos11.pkg  (you'll be prompted for your Mac
        password)
      * Creates a private Python environment in .venv/
      * Downloads required libraries (pandas, customtkinter, etc.)

3. When you see "Setup complete!", close the Terminal.


Step 2.  Run the app  (every time you want to use it)
------------------------------------------------------
1. ** First time only -- bypass Gatekeeper for the launcher: **
      Right-click   run-PowerUp-Portal.command
      Click   Open
      In the warning dialog, click   Open   again
   macOS remembers this choice for this file too.

2. From now on, double-click   run-PowerUp-Portal.command

The PowerUp Portal window opens with 4 tabs (M1 / M2 / M3 / Agreement).


Troubleshooting
---------------
"Operation not permitted" or "Permission denied"
   --> The .command files lost their execute bit during zip extraction.
       Open Terminal, then run:
            cd ~/Documents/PowerUp-Portal-Mac
            chmod +x *.command
       Then double-click again.

Setup says "Installing Python..." then nothing happens
   --> macOS prompted for your password silently. Look for the password
       prompt -- it might be hidden behind Terminal. Type your Mac
       password to allow Python install.

Setup fails -- can't install Python
   --> Double-click python-3.11.9-macos11.pkg yourself, click through
       the installer. When done, re-run install.command.

App opens then crashes
   --> A clear error dialog appears with the full traceback.
       Screenshot and share with Krutang.

Anything else
   --> Ping Krutang with a screenshot.
